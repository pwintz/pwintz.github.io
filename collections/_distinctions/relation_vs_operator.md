Relation vs. operator
Relations and operators both act on one or more objects. The most familiar are “binary relations” and “binary operators”, which act on two objects, so we will describe the differences in terms of these.
* relation: A binary *relation* gives a truth value for each pair of objects. For example, “=“ is a binary relation on real numbers, so “2 = 3” is false and “4=4” is true.
* operand: A binary *operand* maps a pair of objects to a new obect of the same type. Example: addition is a binary operator on numbers, indicated by a plus sign “+”, so “2 + 4” produces a new number, “6”.
