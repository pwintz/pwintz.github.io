## Continuous function vs. open function
Definition: A function $f$ is *continuous* if for each open set $V$, preimage of $V$ under $f$ is open. That is, $f^{-1}(V)$ is open for all open sets $V$ (in the codomain of $f$). 
Definition: A function $f$ is *continuous* if for each open set $U$, image of $U$ under $f$ is open. That is, $f(U)$ is open for each open set $U$ (in the domain of $f$).

**Example** (continuous but not open). Consider $x \mapsto \sin(x)$. If you take $U = [-\pi, \pi]$, then $\sin(U) = [-1, 1]$, which is not open.
