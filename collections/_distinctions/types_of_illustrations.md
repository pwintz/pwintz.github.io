* All of these terms or used informally to mean a visual representation of data or other information. Some have more specific definitions.
* The *graph* of a function $f$ is a set of all points $(x, f(x))$ such that $x$ is in the domain of $f$. If $f$ is a function $\mathbb{R}$ to $\mathbb{R}$, then a graph is a typical way of plotting the function in the Cartesian plane.

* A figure is an element of a document that includes an illustration or picture
* A diagram depicts an abstract representation of relationships.  Examples: Flow chart, feedback diagrams. Non-examples: A graph of a function, scatter plot, pie chart, circuit diagram
