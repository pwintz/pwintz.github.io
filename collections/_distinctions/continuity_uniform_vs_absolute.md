## absolutely continuous ⊂ uniformly continuous
**Example.** (Uniformly continuous but not absolutely continuous). 
- Cantor function.
- This function ([source](https://en.wikipedia.org/wiki/Absolute_continuity#Examples))
$$
f(x)= \begin{cases}0, & \text { if } x=0 \\ x \sin (1 / x), & \text { if } x \neq 0\end{cases}
$$
