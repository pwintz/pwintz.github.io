* "infinitely-many": 
* A set is unbounded (with respect to the metric) if for all M > 0, there exists two points in the set such that the distance between them is greater than M. The definition of "distance" must be defined, so the set must be in a metric space. 
* 
* "infinite set" A set is said to have infinitely many elements if for each M > 0, you can select more than M elements. Equivalently, there exists a surjection onto the natural numbers, which is to say, by assigning each element of the set onto one natural numbers, you can cover the set of natural numbers.
* "infinite set": A set is infinite if it has infinitely many elements.
