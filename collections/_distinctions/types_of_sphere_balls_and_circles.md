Sphere, ball, open ball, closed ball
* A sphere is a set of all points that are exactly a given distance from a point.
* An open ball is a set of all points that are less than a given distance from a point.
* A closed ball is a set of all points that are less than or equal to a given distance from a point.
* A ball is either any open or closed ball.
