Add, union, prepend, append, insert, concatenate, refine, sum, introduce, Minkowski sum
* "refine" indicates adding more elements to a covering
* "add": Has several definitions. 
	* Formal: Addition, indicated by "+" is an binary operation on numbers or (more generally) any two members of a group.
	* Informal: "Add" can be used as a synonym of "insert" as in "an element into a collection". 
* "union":
	* A binary operation on sets that forms a new set that contains all of the elements that are in either set. (Informally, you "add all of the elements of one set into another")
* "prepend"/"append": Indicates placing a new item into an ordered list at the beginning or end (respectively).
* "insert": 
	* Indicates placing an item into an ordered list
	* Indicates placing an item into a set to create a new set.
* "concatenate": A binary operation on two lists. The concatenation of lists A and B is a new list that contains all of the elements of A followed by all of the elements of B.
