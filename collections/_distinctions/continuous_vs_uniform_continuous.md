##  uniformly continuous ⊂ continuous 
Uniform continuity is a stronger condition than continuity, so we write it (informally) as 
	uniformly continuous ⊂ continuous.
More formally, the set of uniformly continuous functions is a subset of the continuous functions. In other words:
- every uniformly continuous function is continuous (as the name suggests)
- some continuous functions are not uniformly continuous.
**Example.** (Continuous but not uniformly continuous). The following functions are continuous but not uniformly continuous:
- $x \mapsto e^x$
- $x \mapsto x^2$
- $x \mapsto \tan(x)$ on $(-\pi/2, \pi/2)$
- $x \mapsto \sin(1/x)$ on $(0, \infty)$

Notes:
- Uniform continuity is defined only for functions between metric spaces, whereas continuity is defined for functions between any topological space.
- To distinguish continuity from uniform continuity, we sometimes call the former "ordinary continuity".
- Uniform continuity is a global property, where is continuity is a local property.
- On a compact domain, every continuous function is uniformly continuous.
