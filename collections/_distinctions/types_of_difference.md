Error, distance, difference, set difference, change, offset.
* All can be used to indicate how far one item is from another (in some sense). 
* "error" indicates the distance of an estimate from   a "true" value.
* "distance" is usually taken to be a nonnegative number, such as "the distance from 
* -2 to 5 is 7". 
* "difference" is a more nebulous term. If used as a technical term, check for a formal definition
