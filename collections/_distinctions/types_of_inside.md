
In, element of, contained within, circumscribed, subset
* "in" and "element of", notated "$\in$" is a relation between an object and a set. An object is in a set if is one of the elements of the set. This is independent of any geometrical relationships. Whereas in every day speech, we may say that "water is in a bottle", we should avoid this usage in mathematics. As an example, consider a sphere and a ball, both centered at a point $c$ with radius $r$. The elements of the sphere all the points that are exactly a distance $r$ from $c$. This means that the center $c$ is not in the sphere (not an element of the sphere), unless $r = 0$. In contrast, the ball contains all of the points with a distance less than or (depending on the definition) $r$  from $c$, so $c$ is an element of the ball.
